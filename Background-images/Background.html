observation

contain: both width and height equal to or exceeds the content area

cover: both width and height will fit inside the content area

background-origin: it represents from where the background-image should start

Background-origin property can have three values 1. padding-box(default) 2. border-box 3.content-box

background-clip: it represents from where the background-color should start

Background-clip property can have three values 1. padding-box 2. border-box(default) 3.content-box

background-origin: content-box;

background-origin: border-box;

background image starts from border

background-clip: content-box;

background color starts from content

background-origin: padding-box;
