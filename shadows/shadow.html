we can give the shadow to element or text

text-shadow--> for applying the shadow to the text

box-shadow--> for applying the shadow to the entire element

text-shadow: 2px 2px;

for text-shadow property we have to give the horizontal and vertical values

text shadow with color

text-shadow: 2px 2px red;

multiple shadows

text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;

box-shadow

box-shadow: 5px 5px 6px gray;

the third value 6px represents the blur value
